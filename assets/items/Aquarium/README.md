# ショップ設備アイテム画像（水槽関連）

axolotl-shop の設備タブで表示するアイテムアイコンを配置します。

## フォルダ構成

```
aquarium/
├── Auto Feeder/    ... Auto_Feeder.png, Copper_Feeder.png, Silver_Feeder.png, Golden_Feeder.png
├── Filter/        ... Sponge Filiter.png, Top Filter.png, Canister Filter.png
├── Food/          ... Copper_Food.png, Silver_Food.png, Golden_Food.png, Diamond_food.png
├── Vacuum/        ... Black_Vacuum.png, Copper_Vacuum.png, Silver_Vacuum.png, Golden_Vacuum.png, Diamond_Vacuum.png
└── Medicine/      ... Medicine.png, Great Medicine.png, Super Medicine.png, Hiper Medicine.png
```
