## 使用方法

### JavaScript

```javascript
// ウーパー画像（新しいパス）
var path = '/assets/characters/axolotl/axo_nomal.png';

// うんこ画像（新しいパス）
var unkoPath = '/assets/items/unko.png';

// 鉱石・宝石
var orePath = '/assets/ores/iron_ore.png';
var gemPath = '/assets/ores/ruby.png';

// 通貨
var coinPath = '/assets/money/coin_32.png';
var billPath = '/assets/money/1000yen.png';

// ヘルパー関数を使用（asset-helper.js読み込み後）
var path = getAxolotlImagePath('nomal');
var unkoPath = getItemImagePath('unko');
```

### HTML

```html
<!-- ウーパー画像（新しいパス） -->
<img src="/assets/characters/axolotl/axo_nomal.png" alt="ウーパー" />

<!-- うんこ画像（新しいパス） -->
<img src="/assets/items/unko.png" alt="うんこ" />
```

## 追加方法

新しい共通アセットを追加する場合：

1. `/assets/` にファイルを配置
2. このファイルに追加内容を記載
3. 必要に応じて `asset-helper.js` にヘルパー関数を追加
