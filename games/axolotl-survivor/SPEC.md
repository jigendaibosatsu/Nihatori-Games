# ウーパーサバイバー - 仕様書

## ゲーム概要
- **タイプ**: バンパイヤサバイバー風サバイバーアクションゲーム
- **テーマ**: ウーパーが敵から逃げながら戦う
- **コンセプト**: 自動攻撃、レベルアップ、大量の敵

## コアシステム

### 1. プレイヤー
- **操作**: 移動のみ（タップ/マウスで移動）
- **自動攻撃**: プレイヤー周囲に自動で攻撃が発生
- **HP**: 初期100、敵に触れるとダメージ

### 2. 敵システム
- **生成**: 画面外から次々と出現
- **AI**: プレイヤーに向かって移動
- **種類**: 複数種類（HP、速度が異なる）
- **難易度**: 時間経過で敵が強くなる

### 3. レベルアップシステム
- **経験値**: 敵を倒すと獲得
- **レベルアップ**: 経験値がたまるとスキル選択
- **スキル種類**:
  - 攻撃力アップ
  - 攻撃速度アップ
  - 移動速度アップ
  - 最大HPアップ
  - 範囲拡大
  - 特殊攻撃（例: うんこ投げ）

### 4. 自動攻撃システム
- **基本攻撃**: プレイヤー周囲に自動で攻撃
- **攻撃タイプ**:
  - 近接攻撃（周囲）
  - 投射物（前方）
  - エリア攻撃（範囲）
- **攻撃速度**: スキルで改善可能

### 5. ゲーム進行
- **時間制限**: 15分間（または無制限）
- **ウェーブ**: 時間経過で敵が強くなる
- **目標**: できるだけ長く生き残る
