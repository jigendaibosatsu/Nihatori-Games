日常更新テンプレ（最短・確実）

Nihatori Games は Cloudflare Pages + Git で運用している静的サイトです。
この手順だけ守れば、必ず反映され、必ず履歴が残ります。

0. 作業開始（毎回）

cd /Users/hondatarou/Desktop/Game/Nihatori-Games
git pull

1. 編集（Cursor）

  • UI：index.html / styles.css / app.js
  • 記事：blog/*.html
  • ゲーム：games/*
  • 一覧データ：data/feed.json
  • 画像：assets/*

2. 反映セット（必ずこの3行）

git add .
git commit -m "update"
git push
注意
・git push だけでは反映されない（commit必須）
・この3行は1セット

3. 反映確認

git status

nothing to commit, working tree clean
が出ていればOK。
Cloudflare Pages の Deployments が Success になれば本番反映完了。

記事を追加する

  1. blog/YYYY-MM-DD-title.html を作成
  2. data/feed.json に1件追加
  3. 反映セット（add / commit / push）

ゲームを追加する

games/slug/
  index.html
  main.js
  assets/

追加後、feed.json に登録 → 反映セット。

トラブル時チェック

  • 反映されない → commit忘れ or Deployments確認
  • 画像が出ない → /assets/ パス確認
  • 表示が古い → ハードリロード（⌘⇧R）

Cursor用・更新プロンプト（保存用）

- 記事は blog/ にHTML1枚
- ゲームは games/slug/
- 一覧制御は feed.json のみ
